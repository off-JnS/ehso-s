Place About assets in this folder.

- MP4 (optional): place `about.mp4` — H.264/MP4 recommended.
- GIF (preferred for autoplay): place `about.gif` — the site references `/videos/about.gif` and will display it as the looping hero.

Recommendations:
- Resolution: 1280×720 or 1920×1080.
- For GIFs: optimize palette and size for web (use lossy GIF optimizers or convert to animated WebP/AVIF if suitable).
